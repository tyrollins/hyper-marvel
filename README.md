# hyper-marvel
> dark & colorful [Hyper](https://hyper.is) theme

Suggested Add-ons:
* font: SF Mono
* term: zsh (OH-MY-ZSH)

## Install
Add `hyper-marvel` to the plugins list in your `~/.hyper.js` config file and restart Hyper.
